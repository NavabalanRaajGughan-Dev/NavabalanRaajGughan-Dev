> *"Where culinary artistry meets cutting-edge web design."*
>
> A premium, cinematic restaurant website built for an award-winning fine dining experience
> in Colombo, Sri Lanka — featuring buttery-smooth animations, parallax storytelling,
> and a fully data-driven architecture.

<br>

<div align="center">

#### 🧬 Architecture

```
╔══════════════╦══════════════╦══════════════╦══════════════╗
║  FRAMEWORK   ║  ANIMATION   ║   SCROLL     ║    STATE     ║

║  App Router  ║  Framer      ║  Smooth      ║  Lightweight ║
║  + React 19  ║  Motion      ║  Engine      ║  & Fast      ║
╠══════════════╩══════════════╩══════════════╩══════════════╣

╚══════════════════════════════════════════════════════════╝
```

</div>

<br>

#### 🎯 What Makes It Stand Out

| | Feature | Description |
|:---:|:---|:---|
| 🎬 | **Cinematic Animations** | GSAP + Framer Motion — parallax reveals, scroll-triggered transitions, elegant flows |
| 🧈 | **Buttery Scroll** | Lenis-powered smooth scrolling that feels native and premium |
| 📂 | **Data-Driven Content** | Menu, gallery, testimonials — all from simple TypeScript config files |
| 📱 | **Pixel-Perfect Responsive** | Crafted for every screen — mobile, tablet, and ultra-wide desktop |
| ⚡ | **Performance First** | Next.js 15 App Router with React 19 for blazing-fast server rendering |
