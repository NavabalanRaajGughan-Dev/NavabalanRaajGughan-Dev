## 🚀 Project Showcase

### 🍽️ [restaurant-project](https://github.com/NavabalanRaajGughan-Dev/restaurant-project)
**Aurum — Fine Dining Restaurant Website**

> A premium, beautifully animated restaurant website template built for a fictional award-winning fine dining experience located in **Colombo, Sri Lanka**.

**✨ Key Highlights:**
- Rich animations powered by **GSAP** & **Framer Motion** (parallax, scroll reveals, smooth transitions)
- Buttery smooth scrolling via **Lenis**
- Fully responsive — mobile, tablet & desktop
- Dynamic content management through centralized TypeScript data files
- Lightweight state management with **Zustand**
